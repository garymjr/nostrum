    all() :: Enumerable.t

    Retrieves a stream of all guilds.

    transform(list) 
    transform(search_criteria_with_transform) :: {:ok, term}

    Retrieves a transformed guild from the cache.

    Returns the result of applying a given fun to the guild specified by the search criteria.

    This transform does not modify the cache and is performed entirely server side. This means that if you somehow cause an error to be thrown, the receiving guild process will die. The benefit is not “copying” guild binaries between processes which can be very big.

    Parameters

    • search_criteria_with_transform - The criteria used to search. See search_criteria_with_transform/0 for more info.

    Example 

    tf = fn guild -> Enum.any?(guild.members, &(Map.get(&1, :nick) == "The year is 20XX")) end
{:ok, plays_fox?} = Nostrum.Cache.Guild.GuildServer.transform(id: 123, transform: tf)
